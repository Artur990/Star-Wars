# Mój projekt:

## Technologie:
	-Next.js, TypeScript, React Query, Axios.
## Funkcjionalności: 
	-Pobranie postaci oraz tytuły filmów z Api.
	-W każdej postaci wyświetlają się filmy w, którj jest postacią.
	-Każdy film posiada możliwość dodania komentarza.
## UI: 
	-Tailwind CSS
## Database: 
	-Node.js
## API:
	https://swapi.dev/api

```bash
npm run dev
```
