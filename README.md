# Mój projekt:

## Technologie:
	-React, TypeScript, Next.js, React Query, axios.
## Funkcjionalności: 
  -Pobranie postaci oraz tytuły filmów z Api.
  -W każdej postaci wyświetlają się filmy w, którj brała udział.
  -Każdy film posiada możliwość dodania komentarza.
## UI: 
	-Tailwind CSS
## Database: 
	-Node.js

```bash
npm run dev
# or
yarn dev
```
