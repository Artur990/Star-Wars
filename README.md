# Mój projekt:

## Funkcjonalności: 
-Pobranie postaci oraz tytuły filmów z Api.
-W każdej postaci wyświetlają się filmy, w której brała udział.
-Każdy film posiada możliwość dodania komentarza.
## Technologie:
	-React, TypeScript, Next.js, React Query, axios.
## UI: 
	-Tailwind CSS
## Database: 
	-Node.js

## How to run project?
```bash
npm run dev
# or
yarn dev
```
